# Settlr

A lightweight balance-tracking app for Android. Log what you're owed, what you owe, and settle up — all in under 5 seconds.

> Currently in active development.

## Stack

- Kotlin + Jetpack Compose
- Material 3 Expressive
- Koin (DI)
- DataStore + kotlinx.serialization
- Min SDK 30 / Target SDK 35

## Features (planned)

- Track balances with people
- Quick-add entries in seconds
- Natural language input
- Activity feed
- Reminders
- Premium tier (coming soon)

## Status

Early development — core skeleton complete, UI polish in progress.
